# Sales-Dashboard-Using-Tableau

This dashboard is a Superstore Sales Dashboard designed to monitor and analyze sales performance, profitability, customer segments, and regional business trends. It provides interactive visual insights that help management make strategic business decisions.
Dashboard Overview

The dashboard presents key business metrics for a retail superstore across:

Sales
Profit
Orders
Product Categories
Customer Segments
States/Regions
Time Trends

The interface includes filters that allow users to dynamically explore performance by:

Measure Type
Region
Year
Top N Products
1. KPI Summary Section

At the top center, the dashboard displays key performance indicators:

Sales
Total Sales: ₹2.3M
Includes a trend sparkline showing sales movement over time.
Profit Margin
Displays overall profitability percentage:
12.47%
Return Rate
Indicates percentage of returned orders:
5.91%

These KPIs help management quickly assess overall business health.

3. Category Wise Profit
